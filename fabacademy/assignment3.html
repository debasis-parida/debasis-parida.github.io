Vinyl Cutting

The Vinyl cutter machine is used to create signs or letters, make banners etc.It is a type of computer-controlled machine. Small vinyl cutters look like a desktop printer. Like a printer controls a nozzle, the computer controls the movement of a sharp blade over the surface of the material. This blade is used to cut out shapes and letters from sheets of thin self-adhesive plastic.

Setup Roland GS-24:

So first i have to setup the Roland GS-24 vinyl cutter by installing the required drivers, softwares and then i shall print a test print into it.Visit https://startup.rolanddg.com/ and select “CAMM-1” series, then select the printer model as “GS-24”. Also select the mode of connection of machine with the system. In my case it is USB connection as shown below.

Next, I downloaded the driver by accepting all the terms and conditions and install the driver following the procedures below:

  • Log on to Windows as “Administrators.”
  • Unzip the downloaded file, and open the folder.
  • Open the folder for your operating system. (WINX64)
  • Double-click the file "SETUP64.exe" (64-bit edition) located in the folder.
  • When "User Account Control" window appears on the screen, click [Continue].
  • Select [Install], "Model" to install, port [USB], and click [Start].
  • When driver is installed successfully, turn on the power switch of the machine, and connect the machine to the computer using the USB cable. Then the computer shall install driver for Roland GS24 and after some time it should appear in “Devices & Printers” menu of Computer.

Download Roland Onsupport

After successfully downloading the driver, then i need to download the roland onsupport application where i can the required manuals and also the "Roland Cutstudio" Software which i will use to cut the vinyl. So i downloaded Roland Cutstudio following the steps and created an account to signing in. Then under tab "Edit->Device Management" i have an option to choose the mode of connection. I chose USB while connecting my Machine to the system USB. Then i selected the COM PORT and click on verify to pair the machine with the system.

Then under "Software" Tab there was a option to download "Roland Cutstudio" which i downloaded and installed.Then i designed a Text of my name in Roland Cutstudio as shown below:(I have also defined the Roll size 23 inch which i used in my machine).

Now, load the Roll in the machine, and Pull the lever in correct position to lock the Sheet. Then using arrows, i can move the pointer to desired position. Then placed the rollers on the White markers and then set the origin by long pressing the origin button.

Then in "Roland Cut studio", Click on cutting, then the preview Tab with all the details came and to start cutting, i had to click on "OK".

After cutting, I used transfer Tape to remove the texts and pasted it on my Table.

Laser Cutting:
